# 鲨域数据库结构知识库

## 文件说明

- 文件名称：`README.md`
- 文件作用：说明鲨域数据库结构知识库的存放规则，以及它与 SAAS / 享佳云 / 鲨域 统一运行目录规则之间的关系。
- 文件主要内容：鲨域数据库结构目录、运行记录目录规则、接口库状态、维护范围和安全规则。
- 所属阶段：接口与数据结构上下文获取 Agent 知识库准备阶段。

## 运行记录目录规则

- 后续用户提到 `鲨域`、`shark`、`SAAS`、`享佳云` 时，新需求运行记录目录统一使用稳定系统标识 `saas`。
- 示例：不再单独生成 `*-shark-*` 运行目录，统一生成 `*-saas-*` 运行目录。
- 用户原始叫法仍需要写入报告说明，例如 `用户指定范围：鲨域`。

## 数据库范围规则

- 数据库结构仍然区分，不随运行目录合并。
- 用户明确说 `鲨域` 时，数据库信息获取 Agent 必须使用本目录下的 `database/`。
- 用户明确说 `SAAS` 或 `享佳云` 时，数据库信息获取 Agent 使用 `knowledge_base/saas/database`。

## 目录说明

- `database/`：保存鲨域相关数据库结构文件。
- `apis/`：当前独立鲨域接口库已删除；如后续需要恢复，需要重新同步对应 Apifox 项目。

## 当前规划

- 数据库：`mall4cloud_saas_*` 系列库和 `scrm`。
- 接口：当前按 `SAAS / 享佳云 / 鲨域` 的统一接口知识库处理；若恢复独立鲨域接口库，需要在任务报告中明确接口来源。

## 安全说明

- 本目录只保存数据库结构信息。
- 不保存数据库账号密码。
- 不导出业务数据。
- 不包含表内记录。
